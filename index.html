Amrit's Story

Born in Nepal, a graduate in Environmental Sciences and now a portuguese citizen,Amrit is building his own silk road to self-development to develop himself to an asset to human society. His Intrest in programming starts from the end of 2017, where he started his journey by learning HTML, css and javascript through a mobile application. It was in in the start of 2018, He comes to know about FreeCodeCamp-Lisbon, a community that meets every week to learn and share coding knowledge. Since then the self learning path became stronger with motivating people around him. He learned Full-stack web development with Javascript everywhere paradigm with tools like: HTML5, CSS3, SASS/SCSS, Javascript/ES6, React.js, Express.Js, MongoDB, Node.Js, JQuery, Bootstrap.

In Septemper 2019, To fulfill his carrer needs and stand as a good programmer, He joined a 14 week intensive and immersive programming bootcamp in Code For All_the Netherlands. There He learned Tools and concepts such as, Object-Oriented Programming, Java, Git, Documentation and Code Conventions, Network Programming, Concurrent Programming, Testing, Debugging, Relational Database Management Systems, Java Database Connectivity, Java Persistence, Spring Framework, Java Web Programming and many more.

In 2020, Amrit is set to start his professional carrer with a company to grow together and to make the human society richer with awesome ideas and technologies.
